# Music_Emotion_Recognition

A system which identifies the emotion of the song with the help of audio sample. Machine Learning algorithms are used to learn and predict the emotion of the song.

## Requirements
1.  jupyter
2.  LibROSA
3.  sklearn
4.  pandas
5.  tkinter
